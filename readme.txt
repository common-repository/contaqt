Integrates the contaqt (article show) Plugin into your WordPress website.

== Description ==
This plugin is an addon service for enrolling to a subscription on contaqt.nl.
It allows you to display articles on your wordpress website. These articles you manage at [Contaqt](http://login.contaqt.net/).
Signup for a demo at: http://contaqt.nl/

Major features in Contaqt include:

* Responsiveness.
* No of articles display on your website.
* Display single article on your website.

PS: You'll need an [Contaqt.nl API key](http://contaqt.nl/) to use it.

== Installation ==

Thank you for your interest in [WordPress Contaqt](http://contaqt.nl/).

Minimum requirements.

* WordPress 4.3+
* PHP 5.x
* MySQL 5.x

Perform a new installation

After downloading the ZIP file,

1. Log in to the administrator panel.
2. Go to Plugins Add > New > Upload.
3. Click "Choose file" ("Browse") and select the downloaded zip file.
4. Click "Install Now" button.
5. Click "Activate Plugin" button for activating the plugin.
6. If the installation does not succeed, please contact us at info@uniekwerk.nl.

Shortcode

1. [article_plugin] : You can show the news articles by pasting the following code in the desired place.
2. [article_plugin_single id="id"] : You can display an individual article by pasting the following code in the desired location.
3. [subscribe_button] : Registration button.
